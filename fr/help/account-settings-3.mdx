---
- title: Paramètres
- description: Tous les élèves Edsby peuvent utiliser les Paramètres pour changer
    leur mot de passe, les Paramètres de notification, le thème de l'interface
    utilisateur et, sur certains systèmes, les préférences linguistiques. Les
    Paramètres permettent également à un élève de voir son nom d'utilisateur,
    son rôle et d'autres informations personnelles.
- sidebarTitle: Aperçu
---

Les élèves peuvent accéder à leurs Paramètres en cliquant sur leur profil dans le coin supérieur droit de la barre de navigation du navigateur et en sélectionnant Paramètres.

<Frame>
  ![](https://www.edsby.com/wp-content/uploads/2021/05/student-browser-settings.png)
</Frame>

Ou, sur un appareil mobile, cliquez sur Paramètres sur l'écran d'accueil.

<Frame />

## Informations de connexion

Affichez le nom d'utilisateur et le mot de passe actuel en cliquant sur Informations de connexion. Les noms d'utilisateur ne peuvent pas être modifiés.

<Frame />

S'il est incorrect ou nécessite une mise à jour, contactez l'école. Cependant, les élèves peuvent changer leur mot de passe. Tapez le nouveau mot de passe et confirmez-le en le tapant à nouveau sous Vérifier le mot de passe. Cliquez sur Enregistrer.

Le nouveau mot de passe est effectif immédiatement.

## Informations personnelles

Les élèves peuvent accéder à leurs Informations personnelles en sélectionnant ceci dans le menu Paramètres. Si ces informations sont incorrectes ou nécessitent une mise à jour, contactez l'école.

<Frame />

Le rôle pour tous les élèves dans Edsby est *Élève.*

## Notifications

Affichez et mettez à jour les Paramètres de notification en cliquant sur Notifications. Ici, les élèves peuvent choisir d'être notifiés concernant leurs Messages, Calendrier personnel, Organisation, Classes, Groupes ou Portfolio.

<Frame />

Cliquez sur une ligne pour la mettre à jour.[Pour plus d'informations sur la modification de vos Paramètres de notification, cliquez ici.](https://www.edsby.com/help/notifications-students-teachers/?locale=en)

## Autres paramètres

Les élèves peuvent modifier leurs préférences linguistiques en cliquant sur Autres paramètres.

<Frame />

<Steps>
  <Step>
    Sélectionnez la langue préférée à l'aide du menu déroulant. Cliquez sur Enregistrer.[Pour plus d'informations sur les paramètres de langue et la traduction dans Edsby, cliquez ici.](https://www.edsby.com/help/language-settings-and-translating-in-edsby/?locale=en#:~:text=In%20the%20row%20next%20to,exit%20the%20Account%20Settings%20menu.)
  </Step>

  <Step>
    Les élèves peuvent également modifier le thème de l'interface Edsby en utilisant le menu déroulant sous Thème de l'interface utilisateur. Essayez le Mode sombre pour une touche amusante !
  </Step>
</Steps>
