---
- title: Visualisation des bulletins scolaires
- description: Si une organisation éducative a choisi de partager les bulletins
    scolaires en ligne via Edsby, les parents peuvent consulter les bulletins
    scolaires de leurs enfants via leur compte Edsby.
---

Si l'organisation partage ses bulletins scolaires en ligne, un bouton deviendra disponible au bas du panneau Classes de chaque enfant. Cliquez sur le bouton Voir les bulletins scolaires. Cela affichera une liste de tous les bulletins scolaires précédents pour cet enfant. Sélectionnez un bulletin scolaire pour le visualiser.

<Frame>
  ![](https://www.edsby.com/wp-content/uploads/2023/09/Edsby-parent-report-card-view.png)
</Frame>

Pour télécharger et enregistrer une copie locale d'un bulletin scolaire, visualisez le bulletin scolaire à partir d'une connexion Edsby sur un navigateur de bureau. Utilisez la fonction d'impression en PDF disponible dans la plupart des navigateurs grand public pour enregistrer le bulletin scolaire dans un fichier PDF sur un appareil. Selon le format du bulletin scolaire, les meilleurs résultats peuvent être obtenus en désactivant les marges et les en-têtes et/ou en sélectionnant l'orientation portrait ou paysage.
