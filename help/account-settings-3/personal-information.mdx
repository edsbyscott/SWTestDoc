title : "Personal information"
description : "Personal Information displays information about a student that only administrators and the student can see. Students can use it to find their Student ID if they forget it. Information found in the Personal Information window includes:"
sidebarTitle : "Personal Information"
---

- Full name
- Student ID
- Address
- Phone numbers
- Parent names
- Locker number

## Accessing Personal Information

Students can access through their Settings. Click the menu icon in the top left on mobile and select Settings.
